#### Some equations
```math
\sin(\alpha) = \sin(\alpha + \tau) \forall \alpha
\cos(\alpha) = \cos(\alpha + \tau) \forall \alpha
\tan(\alpha) = \tan(\alpha + \tau) \forall \alpha
```
```math
\tau^{-1/2} \int_{-\infty}^{\infty} \exp(-1/2 x^2) = 1
```

Harder:
```math
\int_{-\infty}^{\infty} \exp(-x^2) = \sqrt{\pi}
```
